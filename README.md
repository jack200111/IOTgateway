

# 物联IOT网关系统

网关管理后台，基于Vue2、动态修改页面显示的类型、和动态增添页面显示表单的行数

1.项目：此项目initial分支项目，initial初始分支提供自定义框架

优化（封装组件）的项目在develop分支！！！



2.开发步骤：

- 进入项目启动前端起来，操作如下：

```
    npm run serve
```

3.框架样式：不使用UI框架（需要UI框架的可直行修改组件样式），直接vue2组件写样式。

4.提供统一进行axios请求管理



##### data

title：表示页面的标题

type:表示input输入框 text文本

value:值

selected:下拉框选中的值

slot:提示
